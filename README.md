# Investment Preference Analysis

## Project Overview
This Excel-based project analyzes investor preferences using demographic and ranking data.

## Key Features
- Data Cleaning and Standardization
- Data Validation (Yes/No restrictions)
- Conditional Formatting (Color scales & Icon sets)
- Summary statistics using Excel formulas
- Analytical insights from investor behavior

## Tools Used
- Microsoft Excel
- Pivot Tables
- AVERAGE, COUNTIF, MAX, MIN, COUNTA
- Conditional Formatting

## Key Insights
- Identified most preferred investment avenues
- Compared gender-based preferences
- Analyzed age group investment objectives
- Determined dominant savings goals among young investors
